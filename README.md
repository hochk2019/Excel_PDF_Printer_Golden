# Excel & PDF Printer Golden - Công cụ In ấn & Xuất PDF Chuyên Dụng

Ứng dụng được thiết kế tối ưu cho quy trình vận hành kho và xử lý văn bản số lượng lớn tại Golden Logistics. Hỗ trợ in ấn hàng loạt file Excel, Word, PDF và xuất PDF chuyên nghiệp với các tùy chọn linh hoạt.

## Tính năng nổi bật

- **In/Xuất theo trang:** Hỗ trợ chọn trang in linh hoạt (VD: "2" hoặc "1,3,5-7").
- **Xử lý Sheet thông minh:** Tùy chọn in trang chỉ định cho *Trang đầu* hoặc *Tất cả các Sheet* trong file Excel.
- **Biên tập PDF:** Tự động gộp (Merge) nhiều file PDF thành một file duy nhất.
- **Tùy chọn khổ giấy:** Tự động thiết lập khổ giấy A4/A5 cho máy in.
- **Bộ lọc mạnh mẽ:** Lọc file theo tên, ngày sửa đổi, định dạng (Excel/Word/PDF).
- **Tiện ích khác:** 
  - Xem trước file.
  - Quản lý cấu hình in (Preset).
  - Ghi log lịch sử in ấn chi tiết.
  - Tự động lưu cấu hình sau mỗi lần sử dụng.

## Thông tin phiên bản

- **Phiên bản:** V4.1
- **Tác giả:** Hoc HK
- **Yêu cầu hệ thống:** Windows 10/11, Python 3.x (nếu chạy từ source), Microsoft Office (để xử lý Word/Excel).

## Cài đặt (Dành cho Developer)

1. Clone repository:
   ```bash
   git clone https://github.com/hochk2019/Excel_PDF_Printer_Golden.git
   ```
2. Cài đặt thư viện phụ thuộc:
   ```bash
   pip install -r requirements.txt
   ```
   *(Lưu ý: Cần cài đặt thêm `pywin32`, `pypdf`, `tkinter` nếu chưa có)*
